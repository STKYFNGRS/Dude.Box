DUDE.BOX ONCHAIN API REFERENCE
Last Updated: November 29, 2024

COINBASE WALLET SDK

1. Initialization
----------------
```typescript
const sdk = createCoinbaseWalletSDK({
  appName: 'Dude Box',
  appLogoUrl: '/Dude logo 3.jpg',
  darkMode: true,
  preference: {
    options: 'smartWalletOnly' | 'eoaOnly'
  }
});
```

2. Smart Wallet
----------------
Initialization:
```typescript
preference: {
  options: 'smartWalletOnly'
}
```

Methods:
- sdk.getProvider()
- provider.request({ method: 'eth_requestAccounts' })

3. Regular Wallet
----------------
Initialization:
```typescript
preference: {
  options: 'eoaOnly'
}
```

Methods:
- Same as Smart Wallet
- Additional chain switching support

4. State Management
----------------
Provider Events:
- accountsChanged
- chainChanged
- disconnect

Error Handling:
- Connection failures
- User rejections
- Chain switching errors

5. Transaction Management
----------------
Future Implementation:
- Transaction signing
- Gas estimation
- Transaction status
- Receipt handling
